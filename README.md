# emo_insta
A dark, narrative-driven social media platform inspired by emo/goth aesthetics, where users can create profiles, post content, like/comment, and explore interactive story elements. Designed to evoke a moody, immersive experience while still providing standard social networking features.

> **Note:** This repository includes models, views, and templates to demonstrate the app's functionality, but does not include project-level configuration or sensitive backend code.

## Features

- User authentication (create account, log in/out)  
- Create and update profiles with profile images  
- Upload, view, and delete posts with images and captions  
- Like and unlike posts  
- Comment on posts  
- View a grid-style profile feed  
- Responsive design for mobile and desktop  
- Bottom and top navigation bars for easy browsing  

---


## Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS (custom), minimal JavaScript  
- **Database:** SQLite (default for development)  
- **Styling:** Custom CSS, inspired by Instagram and emo aesthetics
